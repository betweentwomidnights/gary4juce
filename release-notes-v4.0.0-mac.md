gary4juce v4.0.0-mac brings the v4.0.0 plugin line to macOS in AU and VST3 form.

This release tracks the current v4 plugin feature set and includes the final recording-buffer startup safety pass so the processor does not drop early blocks when recording begins.

Included DMGs:
- AU
- VST3

Standalone is intentionally omitted from this release while we continue stabilizing it.

Recommended gary4local version:
https://github.com/betweentwomidnights/gary-localhost-installer-mac/releases/tag/v0.1.9

Close your DAW before replacing plugin files with a downloaded update.
