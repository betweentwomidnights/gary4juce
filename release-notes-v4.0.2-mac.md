gary4juce v4.0.2-mac brings the v4.0.2 plugin line to macOS in AU and VST3 form.

This release makes gary4juce's open-source licensing explicit. The project is now released under the GNU Affero General Public License v3.0 only. The plugin now includes an in-plugin About dialog with direct access to the public source and license.

The macOS DMGs now include the applicable license text, third-party notice, and exact Corresponding Source information used for the build. This release does not change any music models, request formats, or backend requirements.

Included DMGs:
- AU
- VST3

Standalone is intentionally omitted from this release while we continue stabilizing it.

Recommended gary4local version:
https://github.com/betweentwomidnights/gary-localhost-installer-mac/releases/tag/v0.1.10

Close your DAW before replacing plugin files with a downloaded update.
