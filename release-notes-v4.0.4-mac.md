gary4juce v4.0.4-mac brings the v4.0.4 plugin line to macOS in AU and VST3 form.

This release brings the Carey LoRA polish from the current mainline build over to macOS. Carey lego mode now supports LoRA usage more broadly, and the in-repo Carey docs now spell out that regular `acestep-v15-base` is still the safer default when no LoRA is loaded.

This release is paired with gary4local mac v0.1.11.

Included DMGs:
- AU
- VST3

Standalone passed internal sanity checks, but it is not part of this public mac plugin release.

Recommended gary4local version:
https://github.com/betweentwomidnights/gary-localhost-installer-mac/releases/tag/v0.1.11

Close your DAW before replacing plugin files with a downloaded update.
